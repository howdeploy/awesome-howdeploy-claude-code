# /deepseek

Отправить запрос к **DeepSeek V3.2** через OpenRouter API.

## Arguments: {{ARGS}}

## Execution

### Шаг 1: Определить нужны ли файлы

Проанализируй `{{ARGS}}`:
- Упоминаются ли файлы, документы, код?
- Есть ли слова: "прочитай", "проанализируй", "файл"?

### Шаг 2: Если нужны файлы — найти их

**ВАЖНО: Не читай содержимое файлов! Только список имён.**

```bash
ls -la <путь>           # Список файлов
ls <путь>/*.pdf         # PDF файлы
ls <путь>/*.{ts,py}     # Код
```

### Шаг 3: Сформировать команду

**Без файлов:**
```bash
~/.claude/scripts/llm-studio/deepseek-query.sh "{{ARGS}}"
```

**С файлами:**
```bash
~/.claude/scripts/llm-studio/deepseek-query.sh --files file.pdf -- "{{ARGS}}"
```

### Шаг 4: Если задача — изменить файл

После получения ответа:
1. Извлеки результат
2. Запиши через Edit/Write tool

## Специализация

- Универсальные задачи
- Большой контекст (164K)
- Дешёвые токены
- Общение, объяснения

## Примеры

**Простой вопрос:**
```
/deepseek Explain how blockchain works
```

**С файлом:**
```
/deepseek Explain what src/main.py does
```
