# /codex

Отправить запрос к **GPT-5.2 Codex** через OpenRouter API.

## Arguments: {{ARGS}}

## Execution

### Шаг 1: Определить нужны ли файлы

Проанализируй `{{ARGS}}`:
- Упоминаются ли файлы, папки, код?
- Есть ли слова: "исправь", "отрефактори", "проверь", "файл", "код"?

### Шаг 2: Если нужны файлы — найти их

**ВАЖНО: Не читай содержимое файлов! Только список имён.**

```bash
ls -la <путь>              # Список файлов
ls <путь>/*.{ts,js,py}     # Код по расширению
```

### Шаг 3: Сформировать команду

**Без файлов:**
```bash
~/.claude/scripts/llm-studio/codex-query.sh "{{ARGS}}"
```

**С файлами:**
```bash
~/.claude/scripts/llm-studio/codex-query.sh --files src/file.ts -- "{{ARGS}}"
```

### Шаг 4: Если задача — изменить файл

После получения ответа от Codex:
1. Извлеки код из ответа
2. Запиши через Edit/Write tool
3. Сообщи что файл обновлён

## Специализация

- Генерация и рефакторинг кода
- Дебаг и поиск ошибок
- Code review
- Оптимизация

## Примеры

**Без файлов:**
```
/codex Write a sorting function in Python
```

**С файлами:**
```
/codex Find bugs in src/utils.ts
```
